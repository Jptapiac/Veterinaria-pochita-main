# Imágenes del Carrusel

Coloca aquí las siguientes imágenes para el carrusel de la página de inicio:

1. **banner-veterinaria.jpg** - Banner promocional de "Veterinarian Clinic" con múltiples perros
2. **clinica-veterinaria.jpg** - Imagen de la clínica veterinaria con veterinarios y animales

Las imágenes deben tener:
- Formato: JPG o PNG
- Resolución recomendada: 1920x500px o similar (proporción 16:5)
- Tamaño de archivo: Optimizado para web (máximo 500KB por imagen)

El carrusel se mostrará automáticamente en la página de inicio una vez que las imágenes estén en esta carpeta.
